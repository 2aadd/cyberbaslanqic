# CyberBaşlanğıc

**Kiber Təhlükəsizliyə Tam Bələdçi**

Azərbaycanca yazılmış, açıq mənbə və tam pulsuz kiber təhlükəsizlik kitabı.

---

## Bu kitab nədir?

Kiber təhlükəsizliyə başlamaq istəyəndə Azərbaycanca normal bir qaynaq tapmaq demək olar ki mümkün deyil. Hər şey ingiliscə idi, hər şey çətin görünürdü. Bu kitab elə o vaxtki mənə yazılmış bir məktub kimidir — sadə dildə, praktiki nümunələrlə, addım-addım.

Kitabı bitirəndə "haker" olmayacaqsan, amma internetin arxasında nə baş verdiyini başa düşəcəksən, terminalda itib qalmayacaqsan və kiber təhlükəsizlik vakansiyasına baxanda "bunları bilirəm" deyəcəksən.

## Necə istifadə etməli?

1. **Ardıcıl oxu.** Networking-i keçmədən Linux-a keçmə — hər şey bir-birinin üzərində qurulur.
2. **Tapşırıqları et.** Oxumaq ayrı şeydir, əllərinlə etmək ayrı şey. Bir əmri terminalda yazmadan öyrəndim sayma.
3. **Sual ver.** Aşağıdaki topluluğa qoşul, çəkinmədən yaz.
4. **Tələsmə.** Kiber təhlükəsizlik sprint deyil, marathon-dur.
